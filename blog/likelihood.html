Notes on Likelihood

Take a Gaussian probability distribution f(x;μ,σ)=σ2π​1​exp(−21​(σx−μ​)2)

This is a function of x and is parameterized by μ and σ. If we instead consider the expression on the right as a function of μ and σ, parameterized by x, we have the likelihood function L(μ,σ;x)=σ2π​1​exp(−21​(σx−μ​)2)

This is the definition of the likelihood function.

I’ll emphasize that the expression on the right hasn’t changed. A few things fall out of this 1) Previously the integral of f (holding μ, σ constant) over x was 1. Now if we integrate L over μ and σ (holding x constant) there is no reason to expect the integral is still 1. 2) L evaluated at (μ=a,σ=b;x=5) is a likelihood. If f happened to be parameterized by μ=a,σ=b and evaluated at x=5 the result would be the same (since the expression is the same). Therefore, the value of f evaluated at x=5 is the likelihood of its parameters parameterized by x=5. That’s a long-winded way to say f(x) is a likelihood. (Remember it is the integral of f which is a probability: ∫ab​fdx).

So what does it mean? Look at the line on this surface for μ=5.5. Notice that near σ=0 it has a low likelihood. As σ increases, its likelihood increases for a bit and then declines again. Consider three points along that line (μ, σ) = (5.5, 0.2), (5.5, 1), (5.5, 2). How did we calculate the value of L at each of these points? We just plugged in these (μ,σ) pairs along with x=5 into the expression above. Let’s visualize our doing that, but letting x vary again (notice these curves are PDFs now).

The value of our likelihood surface for each combination (μ,σ) is the intersection of the red line with each of these PDFs. Orange performs the best. Even if we centered the green PDF on x=5 it would still not have as high a value as the orange curve. So we can think about the values of the likelihood function like this: “how well does this pair of parameters (μ,σ) fit the data point x=5?”

That brings us to Wikipedia’s definition: “A likelihood function (often simply called the likelihood) measures how well a statistical model explains observed data by calculating the probability of seeing that data under different parameter values of the model.” Hopefully this is clear now.

One last thing. Wikipedia’s article continues “It is constructed from the joint probability distribution of the random variable that (presumably) generated the observations”. When I said earlier that the definition of the Likelihood was simply the PDF with its parameters considered as variables that was only partially true. It is actually the joint PDF. In this case we just considered a “joint” PDF with only one part. L(μ,σ;x1​,x2​,...xn​)≜i=1∏n​f(xi​;μ,σ)=i=1∏n​σ2π​1​exp(−21​(σxi​−μ​)2) The likelihood function is useful because it helps us guess the parameters of a distribution given some samples from it. Say we have IID samples x1​,x2​,⋯,xn​, from a known distribution with unknown parameters, and we want to find the parameters which maximize the probability of having drawn this sample. The parameters which maximize the likelihood function are those parameters! If this isn’t clear, think back to the one dimensional case and the previous figure.
